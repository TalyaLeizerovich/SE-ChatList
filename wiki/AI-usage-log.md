Case 1: Writing Automated Tests Using Selenium
----------------------------------------------

**Problem**  
During the system testing phase, we were required to write end-to-end tests for the user interface. We had no prior experience with Selenium or browser-based test automation, and it was unclear how to set up the environment, write basic test scripts, and execute them correctly.
**AI Tool Used**  
ChatGPT, Claude, Gemini
**Main Prompt**

“Explain how to write Selenium tests in Python from scratch, including environment setup, browser drivers, and a simple end-to-end test example.”

**How the AI Output Advanced the Project**  
The AI provided a clear, step-by-step explanation of how to work with Selenium, including installing the required libraries, using WebDriver, locating elements on a web page, and writing basic test scenarios. In addition, we received code examples that could be directly adapted to our system. As a result, we were able to independently write automated tests, improve test coverage, and increase the overall quality of the product.

Case 2: Integrating with External Services (WhatsApp Web and Google Calendar)
-----------------------------------------------------------------------------

**Problem**  
Our system needed to integrate with external services such as WhatsApp Web for reading messages and Google Calendar for automatically creating events. We had no prior experience working with these services, particularly regarding browser automation, APIs, authentication, and permission management.
**AI Tool Used**  
ChatGPT, Claude, Gemini
**Main Prompt**

“How can a Python backend integrate with WhatsApp Web and Google Calendar? Explain possible approaches, limitations, authentication, and example flows.”

**How the AI Output Advanced the Project**  
The AI presented several possible approaches for integrating with the external services, explained the differences between using official APIs and browser automation, and highlighted security and maintenance considerations for each option. It also provided conceptual flow descriptions and code examples. This guidance helped us choose a solution that fit the project requirements, design the system architecture correctly, and avoid architectural mistakes early in development.

Case 3: Consulting on Models for Text Processing and Task Extraction
--------------------------------------------------------------------

**Problem**  
The system includes a component for processing free-text messages and extracting actionable tasks. We debated which approach to use: rule-based logic, classical NLP models, or large language models (LLMs), and what the trade-offs of each approach would be in the context of our project.
**AI Tool Used**  
ChatGPT, Claude, Gemini
**Main Prompt**

“Compare rule-based approaches, classical NLP models, and large language models for task extraction from chat messages. When should each approach be used?”

**How the AI Output Advanced the Project**  
The AI provided a clear comparative analysis of the different approaches, addressing aspects such as accuracy, computational cost, maintenance effort, scalability, and dependency on external services. These insights enabled us to make an informed decision regarding model selection and to consider a hybrid approach that combines rule-based logic with a language model, in accordance with the project’s constraints and goals.
